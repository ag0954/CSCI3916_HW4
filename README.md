### Installing

1. Clone the repository
```zsh
git clone 
```

1. Install dependencies
```zsh
cd current_Repository
npm install
```

1. Start the server
```zsh
npm start
```

The server will start on `localhost:8000`.

## Running the tests

To run the tests, use the following command:

```bash
npm test
```
